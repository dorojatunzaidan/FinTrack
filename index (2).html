Panduan Setup Google Sheets
1. Buka console.cloud.google.com
2. Buat project baru → Aktifkan Google Sheets API
3. OAuth consent screen → External → Tambah scope Sheets
4. Credentials → Create → OAuth 2.0 Client ID → Web Application
5. Authorized JS origins: tambahkan URL domain tempat kamu hosting file ini
6. Copy Client ID → paste di field di atas
7. Share Google Sheet kamu dengan "Anyone with link (Viewer)"
